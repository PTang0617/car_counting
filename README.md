# Car Counting with YOLOv8

Using yolov8l to detect the car class and use the 'sort' tracking algorithm to tracking the car. Finally, draw a small roi to count total passed car's amount.

## Set up the environment

`pip install -r requirements.txt`

## Files infos
* `car_counting.py`: main code
* `sort.py`: tracking algorithm
